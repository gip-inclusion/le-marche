API du Marché de l'inclusion

L'API met à disposition le répertoire qualifié des structures d'insertion par l'activité économique.

Conditions générales d'utilisation

Lire les conditions générales d'utilisation de l'API du marché de l'inclusion.

Documentation

Une documentation est disponible pour explorer et tester les possibilités de l'API.

📚 Accéder à la documentation

L'API est aussi publiée sur api.gouv.fr.

Authentification

Certaines données nécessitent un token (jeton utilisateur).

Comment obtenir un token ?

  1. Créez-vous un compte sur le marché de l'inclusion
  2. Puis rendez-vous sur le formulaire de contact pour nous en faire la demande
  3. Une fois généré, le token apparaîtra dans votre espace utilisateur

Comment se servir du token ?

Rien de plus simple, il suffit de rajouter Bearer <VOTRE-TOKEN> dans le header "authorization" de vos requêtes API
